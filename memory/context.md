# Project Context: SwMaster Command Center

## Overview
* **Goal:** A senior software engineering agent framework based on SWEBOK v4, providing an autonomous command center for architecture, construction, and QA.
* **Current Phase (SWEBOK):** Construction Phase (Implementing UI/UX and Agent Orchestration).
* **Tech Stack:** Next.js 16 (React, Tailwind, MD3), FastAPI (Python 3.12), LangGraph (State Machine).

## Current State (Milestone 1: Command Center)
* **Active Objectives:** 
  1. [x] Scaffold Backend & Frontend (Phase 1-2)
  2. [x] Implement LangGraph state machine with SOD (Phase 3)
  3. [/] Phase 4: Thread Persistence, Rich UI, Live KPIs
* **Issues in Focus:** Phase 4 - Advanced Agent Interaction.

## Project-Specific Guidelines
* **Segregation of Duties (SOD):** Strictly enforced at the architectural level. Planner, Maker, and Checker must remain isolated.
* **Aesthetics:** Use Aura Strategy design (MD3, Glassmorphism, Micro-animations).
* **SWEBOK Alignment:** All engineering decisions must cite SWEBOK principles.
